**bxSlider - Plugin jQuery para criar slider responsivo do tipo carousel.**

Olá, como estamos? Hoje quero te apresentar o bxSlider, um plugin jquery para criar slider de imagens responsivo bem fácil de ser utilizado. Já utilizei em meus projetos e achei bem fácil. Vale a pena ter ali nos favoritos, outro detalhe é que ele serve para criar slider carousel tanto horizontal como vertical.

Documentação e instruções: **[http://bxslider.com/](http://bxslider.com/)**
